# mem-66 乡村道路经验保护单元

> 所属仓库: AD-mlnf-mem/spec/
> 模块编号: mem-66
> 所属分区: AD新增·双漏斗管理层


## 一、基本信息

| 项目 | 内容 |
|:---|:---|
| 模块编号 | mem-66 |
| 模块名称 | 乡村道路经验保护单元 |
| 核心职责 | 对乡村非铺装道路的通行经验执行差异化遗忘保护策略。遗忘阈值较常规场景下调50%，经验条目标记“乡村道路”特殊标签，确保关键经验不被过早清除 |
| 输入数据 | 经验条目数据（来自mem-63路由到通用驾驶槽的乡村道路经验）、当前遗忘策略参数（来自mem-28遗忘阈值判定单元）、场景类别标签（来自mem-63） |
| 输出数据 | 调整后的遗忘阈值参数、经验保护标签写入确认、保护策略执行日志 |


## 二、核心处理逻辑

**1. 乡村道路经验识别**

当mem-63将经验条目路由到“通用驾驶槽”时，mem-66检查该经验条目的场景特征。若场景特征中包含以下任一条件，则标记为“乡村道路经验”：

- 路面类型=泥土/碎石/沙土等非铺装路面
- 道路等级=乡村道路/非等级道路
- 无车道标线且道路宽度<3.0m
- 周边环境特征=农田/牧场/林地（来自世界模型mem-59）
- 500m内无交通信号灯且无交通标识

**2. 遗忘阈值差异化调整**

对标记为“乡村道路经验”的条目，mem-66向mem-28（遗忘阈值判定单元）输出调整后的遗忘参数：

- 遗忘重要度阈值 θ_forget：较常规场景下调50%（即常规阈值×0.5）
- 最低复用次数阈值 c_min：较常规场景下调50%（即常规阈值×0.5，向下取整，最低为1）
- 晋升时间阈值 τ：不变（晋升条件中的时间门槛不做差异化调整）

**阈值调整示例**：常规场景下，一条经验若 I < 0.08 且复用次数 < 2，则触发遗忘。乡村道路场景下，同一条经验若 I < 0.04 且复用次数 < 1，才触发遗忘。这意味着乡村道路经验被遗忘的难度翻倍。

**3. 经验保护标签维护**

每条标记为“乡村道路经验”的条目，在其记忆元数据中追加特殊标签 `protection: rural_road`。该标签伴随经验条目的完整生命周期，不会因经验晋升而自动移除。当经验从L1逐层晋升至L5时，保护标签始终保留。若经验在L3/L4层被抽象提炼为通用规则（由mem-14执行），提炼后的规则自动继承保护标签。

**4. 保护策略执行监控**

mem-66持续监测乡村道路经验在各层级的分布情况。若任一场景分槽中乡村道路经验的占比超过该槽总容量的30%，系统自动触发保护策略审计，确认遗忘阈值调整是否被正确应用。审计结果写入mem-55日志系统。

**5. 存储容量管控**

乡村道路经验的保护策略不改变全局容量管控规则（mem-40/41）。当通用驾驶槽的存储容量逼近上限时，遗忘策略依然生效，但乡村道路经验因阈值下调50%而具有更高的保留优先级。容量极端紧张时（槽容量>95%），首先清理常规经验中最低重要度的L1-L2条目，乡村道路经验同等条件下最后被清理。


## 三、约束与异常处理

| 约束类型 | 内容 |
|:---|:---|
| 阈值调整范围 | 遗忘阈值下调50%为固定值，不可通过运行时参数修改 |
| 保护标签持久性 | 保护标签伴随经验完整生命周期，晋升不自动移除 |
| 规则继承 | 经验被抽象提炼为通用规则后，新规则自动继承保护标签 |
| 审计触发 | 乡村道路经验占比>30%时触发保护策略审计 |
| 容量管控 | 乡村道路经验不享受绝对免遗忘保护，仅在同等条件下优先保留 |
| 场景误判 | 若世界模型将铺装道路误判为非铺装道路，保护策略仍然执行；场景判定由mem-63负责，mem-66不参与判定 |


## 四、接口校验用例

**用例1：乡村道路经验识别与标记**

前置条件：mem-63路由一条经验到通用驾驶槽，场景特征包含路面类型=碎石路面，道路宽度=2.5m，无车道标线，500m内无交通标识。操作：mem-66检查该经验条目的场景特征。预期：经验被标记为“乡村道路经验”，元数据中追加保护标签 `protection: rural_road`。

**用例2：遗忘阈值下调验证**

前置条件：一条乡村道路经验的当前重要度 I=0.05，复用次数=1。常规遗忘阈值 θ_forget=0.08，c_min=2。操作：mem-66向mem-28输出调整后的遗忘参数（θ_forget=0.04，c_min=1）。预期：该经验在常规策略下本应被遗忘（0.05<0.08且1<2），但在保护策略下免于遗忘（0.05>0.04且1≥1）。

**用例3：保护标签随晋升保留**

前置条件：一条L2层的乡村道路经验标记了保护标签。操作：该经验满足晋升条件（T>τ且I>θ），由mem-26/27执行晋升至L3层。预期：晋升后该经验的保护标签依然存在，未被自动移除。

**用例4：规则继承保护标签**

前置条件：L3层存在多条标记了保护标签的乡村道路经验（均为不同路段的窄路会车经验）。操作：mem-14（经验抽象提炼单元）将这些经验提炼为通用规则“窄路会车：优先搜索避让点，下坡车让上坡车”。预期：新提炼的通用规则自动继承保护标签 `protection: rural_road`。

**用例5：容量管控优先级验证**

前置条件：通用驾驶槽容量使用率已达96%，触发紧急清理。槽内包含常规经验和乡村道路经验。操作：mem-30（冗余记忆删除/归档单元）执行清理。预期：首先清理常规经验中I值最低的L1条目；乡村道路经验中的低I值L1条目在同等条件下最后被清理。乡村道路经验不享受绝对免遗忘保护。
