# ad-15-高速巡航槽 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-15 |
| 模块名称 | 高速巡航槽 |
| 所属分区 | 三、漏斗二：自动驾驶自成长漏斗 / 场景分槽管理 |
| 核心职责 | 承载高速公路场景下跟车、变道、匝道汇入等驾驶经验的完整五层存储与晋升管理。执行专属遗忘策略：复用频次权重上调 20%，强化高频操作的经验保留优先级。是本场景经验从 L1 临时层到 L5 核心层的完整生命周期管理者 |
| 依赖模块 | ad-14（场景判定与分槽路由单元，接收经验写入指令）、ad-20 至 ad-30（五层存储与晋升遗忘执行模块）、ad-36（综合重要度 I 值聚合计算单元） |
| 被依赖模块 | ad-14（上报存储占用率与活跃状态）、ad-03（漏斗二专属调度单元，接收状态汇总） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常运行 | `NORMAL` | 正常接收经验条目并执行五层存储流转 | 漏斗二已激活，本槽已初始化 |
| 容量告急 | `CAPACITY_WARNING` | 存储占用率 > 90%，已触发加速遗忘机制 | 本槽存储占用率监测 |
| 维护中 | `MAINTENANCE` | 正在执行相似经验归并或冷归档操作 | 定期维护周期或容量告急触发 |
| 冻结只读 | `FROZEN` | 漏斗二冻结或紧急接管，暂停写入但保留查询 | 驾驶模式切换或紧急熔断 |


## 输入数据

| 输入项 | 数据类型 | 来源模块 | 触发条件 | 优先级 |
|--------|----------|----------|----------|:---:|
| 经验写入指令 | Struct（含经验条目 + 写入优先级 + 路由标记 + 判定置信度） | ad-14 场景判定与分槽路由单元 | 每次场景路由至本槽时 | **高** |
| L1 至 L4 晋升候选清单 | List（经验条目 ID + 晋升目标层级 + 重要度 I） | ad-38 晋升双条件判定单元 | 每个晋升判定周期 | **高** |
| 遗忘候选清单 | List（经验条目 ID + 当前层级 + I 值） | ad-40 遗忘阈值判定单元 | 每个遗忘判定周期 | **高** |
| 归并建议 | Struct（源经验 ID + 目标经验 ID + 相似度） | ad-25 L3 中期层相似经验归并单元 | L3 归并操作时 | 普通 |
| 全局调度指令 | Enum（冻结/激活/维护） | ad-03 漏斗二专属调度单元 | 模式切换时 | **高** |


## 输出数据

| 输出项 | 数据类型 | 目标模块 | 输出条件 | 优先级 |
|--------|----------|----------|----------|:---:|
| 存储完成回执 | Struct（经验条目 ID + 存储层级 + 存储时间戳） | ad-14 | 每次经验成功存储后 | 普通 |
| 晋升执行结果 | Struct（经验条目 ID + 原层级 + 目标层级 + 执行结果） | ad-38、ad-39 | 晋升完成后 | **高** |
| 遗忘执行结果 | Struct（经验条目 ID + 删除/归档 + 结果） | ad-40、ad-42 | 遗忘执行后 | 普通 |
| 槽状态快照 | Struct（存储占用率 + 各层级条目数 + 活跃度） | ad-14、ad-03 | 每 1 秒周期性上报 | 普通 |
| 容量告警 | Struct（当前占用率 + 触发阈值 + 建议动作） | ad-03、ad-01 | 占用率 > 90% 时 | **高** |


## 专属遗忘策略

| 策略项 | 标准值 | 本槽调整 | 调整理由 |
|--------|:---:|:---:|----------|
| 复用频次权重（γ） | 0.3 | **0.36（上调 20%）** | 高速场景强调标准化操作的重复执行，复用的经验价值更高 |
| 安全显著性权重（α） | 0.5 | 0.5（不变） | — |
| 风格匹配度权重（β） | 0.2 | 0.14（相应下调） | 权重总和保持 1.0 |
| 最低重要度遗忘阈值 | 0.15 | 0.12（下调 20%） | 高速场景经验即使重要度稍低也优先保留，因为高速事故代价极高 |
| L1→L2 晋升时间阈值 | 24h | 24h（不变） | — |
| L2→L3 晋升时间阈值 | 7日 | 5日（缩短） | 高速经验需更快固化，因场景重复性高 |


## 核心处理逻辑

```
FUNCTION highway_cruise_slot_main_loop():
    SET internal_state = NORMAL
    初始化五层存储结构（L1 至 L5，按漏斗二标准容量比例分配）
    加载专属遗忘策略参数（γ=0.36，最低遗忘阈值=0.12，L2→L3 加速晋升）

    WHILE 系统运行中:
        // 第 1 步：全局调度指令
        IF 收到 ad-03 全局调度指令:
            IF 指令 == 冻结:
                SET internal_state = FROZEN
                CONTINUE
            ELSE IF 指令 == 激活:
                SET internal_state = NORMAL
                CONTINUE

        // 第 2 步：接收并存储新经验
        IF 收到 ad-14 经验写入指令 AND internal_state == NORMAL:
            经验条目 = 写入指令.经验条目
            将经验条目写入 L1 临时层存储分区
            记录写入时间戳
            向 ad-14 返回存储完成回执（层级=L1）

        // 第 3 步：处理晋升候选
        IF 收到 ad-38 晋升候选清单:
            FOR EACH 候选条目 IN 晋升候选清单:
                IF 候选条目.目标层级存在 AND 该层级存储未满:
                    调用 ad-39 执行层级单向搬运
                    IF 晋升成功:
                        向 ad-38 返回晋升执行结果（成功）
                    ELSE:
                        向 ad-38 返回晋升执行结果（失败，原因=存储满）
                ELSE:
                    向 ad-38 返回晋升执行结果（失败，原因=层级不存在）

        // 第 4 步：处理遗忘候选
        IF 收到 ad-40 遗忘候选清单:
            FOR EACH 候选条目 IN 遗忘候选清单:
                IF 候选条目.层级 IN [L4, L5]:
                    SKIP  // L4/L5 不参与遗忘
                ELSE:
                    调用 ad-42 执行安全删除
                    IF 候选条目.层级 == L3:
                        可选：压缩归档至冷存储（ad-49）
                    向 ad-40 返回遗忘执行结果

        // 第 5 步：定期归并相似经验（L3 层）
        IF 距上次归并 >= 24 小时 OR 容量占用率 > 90%:
            SET internal_state = MAINTENANCE
            向 ad-25 发送 L3 相似经验归并请求
            WAIT ad-25 返回归并建议
            IF 存在归并建议:
                执行归并（保留重要度更高的条目，合并复用计数）
                释放冗余存储空间
            SET internal_state = NORMAL

        // 第 6 步：容量监控
        当前占用率 = 计算本槽总存储占用 / 本槽配额
        IF 当前占用率 > 0.90 AND internal_state == NORMAL:
            SET internal_state = CAPACITY_WARNING
            向 ad-03、ad-01 上报告警
            自动触发 L1/L2 加速遗忘（遗忘阈值临时上调 30%）
        ELSE IF 当前占用率 < 0.70 AND internal_state == CAPACITY_WARNING:
            SET internal_state = NORMAL
            恢复标准遗忘阈值

        // 第 7 步：周期性状态上报
        IF 距上次上报 >= 1 秒:
            向 ad-14、ad-03 上报槽状态快照

        SLEEP 100ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| L1 存储分区满 | 触发 L1 强制时序衰减，保留时间 < 1 小时且 I 值最低的条目立即清除 | L1 存储占用 < 80% |
| L3 归并时发现相似度 < 0.6 | 取消归并，两条经验各自保留 | — |
| 本槽配额用完且无法释放 | 上报告警至 ad-03，请求扩容或跨槽迁移至通用驾驶槽 | 手动调整配额或通用驾驶槽接收 |
| 晋升候选条目指向的目标层级不存在 | 跳过该条目，上报告警 | 系统修复 |
| 冻结状态下收到经验写入 | 返回拒绝写入回执（原因=冻结），写入方（ad-14）负责暂存 | 收到激活指令 |
| 容量告警时加速遗忘导致重要经验被误删 | 加速遗忘仅针对 L1/L2 层，L3 及以上受保护 | — |
| 专属遗忘策略参数被异常修改 | 回退至编译期默认值，上报告警 | 参数校验通过后恢复 |


## 总线契约

| 总线 | 操作 | 数据内容 | 权限 | 说明 |
|------|------|----------|------|------|
| MemoryBus | 读 | 经验写入指令 | 只读 | 由 ad-14 下发 |
| MemoryBus | 读 | 晋升候选清单 | 只读 | 由 ad-38 下发 |
| MemoryBus | 读 | 遗忘候选清单 | 只读 | 由 ad-40 下发 |
| MemoryBus | 读 | 归并建议 | 只读 | 由 ad-25 下发 |
| MemoryBus | 读 | 全局调度指令 | 只读 | 由 ad-03 下发 |
| MemoryBus | 写 | 存储完成回执 | 专属写入 | 向 ad-14 返回 |
| MemoryBus | 写 | 晋升执行结果 | 专属写入 | 向 ad-38、ad-39 返回 |
| MemoryBus | 写 | 遗忘执行结果 | 专属写入 | 向 ad-40、ad-42 返回 |
| MemoryBus | 写 | 槽状态快照 | 周期性写入 | 每 1 秒向 ad-14、ad-03 上报 |
| MemoryBus | 写 | 容量告警 | 事件触发写入 | 向 ad-03、ad-01 上报 |


## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | L5 核心层经验物理锁定，仅可读取，不可修改或删除，写入须经 ad-30 安全仲裁验证 |
| S-02 | 专属遗忘策略参数（γ=0.36、最低遗忘阈值=0.12）编译期固化为默认值，运行时可调但不可超出安全边界（γ ≤ 0.5，遗忘阈值 ≥ 0.08） |
| S-03 | 容量告警时加速遗忘仅针对 L1/L2 层，L3 及以上层级受保护，确保中期以上经验不因临时容量波动被清除 |
| S-04 | L5 层不可抗力事件（如高速突遇障碍物成功避险）永久锁定，终身不可遗忘 |
| S-05 | 所有晋升、遗忘、归并操作全量写入 ad-51 变更日志 |
