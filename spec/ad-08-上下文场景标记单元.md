# ad-08-上下文场景标记单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-08 |
| 模块名称 | 上下文场景标记单元 |
| 所属分区 | 二、漏斗一：驾驶员画像漏斗 |
| 核心职责 | 同步获取世界模型环境特征，为每条驾驶行为观测条目打上场景标签（常规路况 / 紧急突发 / 特殊场景）。应急避险操作自动标记为“应急特殊操作”，不纳入陋习统计。场景标签随行为条目写入目标子画像槽，用于后续行为判定 |
| 依赖模块 | ad-44（独立世界模型库，提供道路属性与场景特征）、ad-07（驾驶行为观测记录单元，消费其行为观测条目） |
| 被依赖模块 | ad-09（行为判定标签单元，消费带场景标签的行为条目） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常运行 | `NORMAL` | 正常接收行为观测条目并打标 | 系统初始化完成，世界模型可用 |
| 世界模型查询中 | `QUERYING` | 正在向世界模型查询当前场景特征 | 收到行为观测条目后触发 |
| 场景标签匹配中 | `MATCHING` | 正在将世界模型返回的场景特征与标签规则库进行匹配 | 世界模型返回结果后 |
| 降级默认标记 | `FALLBACK` | 世界模型暂时不可用，使用默认场景标签 | 世界模型超时或返回异常 |
| 暂停标记 | `PAUSED` | 漏斗一冻结或驾驶模式切换，暂停处理 | 驾驶模式切换为非人工驾驶 |
| 紧急只读 | `EMERGENCY_RO` | 紧急事件触发，暂停全部写操作 | 收到 ad-01 紧急熔断指令 |


## 输入数据

| 输入项 | 数据类型 | 来源模块 | 触发条件 | 优先级 |
|--------|----------|----------|----------|:---:|
| 行为观测条目 | Struct（含全部观测字段，详见 ad-07 输出详表） | ad-07 驾驶行为观测记录单元 | 每次行为观测条目生成后 | **高** |
| 世界模型场景特征 | Struct（道路等级、路面类型、天气、光照、时段、交通流密度、周边目标分布、特种车辆检测、碰撞风险等级） | ad-44 独立世界模型库 | 每次查询请求后返回 | **高** |
| 驾驶模式信号 | Enum（人工驾驶/自动驾驶/紧急接管） | ad-02 漏斗一专属调度单元 | 驾驶模式变更时 | **高** |
| 紧急熔断指令 | Enum（全局熔断） | ad-01 总控漏斗 F₀ | 安全急停或碰撞事件时 | **紧急** |


## 输出数据

| 输出项 | 数据类型 | 目标模块 | 输出条件 | 优先级 |
|--------|----------|----------|----------|:---:|
| 带场景标签的行为条目 | Struct（原观测条目 + 场景标签 + 标签置信度 + 应急判定） | ad-09 行为判定标签单元 | 场景标签匹配完成后 | **高** |
| 世界模型查询异常告警 | Struct（异常类型 + 持续时长） | ad-01 总控漏斗 F₀ | 世界模型超时或返回异常超过 3 次 | 普通 |
| 标记状态上报 | Struct（当前状态 + 近期标记统计） | ad-01 总控漏斗 F₀ | 每 5 秒周期性上报 | 普通 |

**场景标签详表：**

| 标签类型 | 标签名称 | 判定依据 | 对行为判定的影响 |
|----------|----------|----------|------------------|
| 常规 | 常规路况 | 道路通畅、天气正常、无突发事件、无特种车辆 | 按标准规则判定 |
| 特殊 | 拥堵蠕行 | 交通流密度 > 0.8，平均车速 < 10km/h | 变道/跟车行为放宽判定 |
| 特殊 | 恶劣天气 | 雨/雪/雾/沙尘，能见度 < 500m | 加减速行为放宽判定 |
| 特殊 | 夜间无灯 | 时段=夜间，路面照度 < 10 lux | 车速行为结合路况判定 |
| 特殊 | 非铺装道路 | 路面类型 ∈ {泥土/碎石/沙土} | 全部行为结合路面条件判定 |
| 紧急 | 碰撞高风险 | TTC < 3s，或碰撞风险等级 ≥ 高 | 触发应急特殊操作判定 |
| 紧急 | 特种车辆临近 | 检测到救护车/消防车/警车声光信号 | 强制让行，全部操作标记应急 |
| 紧急 | 路面突发异常 | 坑洼/塌陷/落石/积水/结冰突然出现 | 避让/制动操作标记应急 |
| 紧急 | 行人/动物突然侵入 | 第四类/第三类目标 TTC < 2s | 全部紧急操作标记应急 |
| 未知 | 未知道路 | 世界模型无法确定场景类别 | 按保守规则判定，标记低置信度 |


## 核心处理逻辑

```
FUNCTION context_tagging_main_loop():
    SET internal_state = NORMAL

    WHILE 系统运行中:
        // 第 1 步：最高优先级——紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = EMERGENCY_RO
            暂停全部标记处理
            丢弃未完成的标记任务
            CONTINUE

        // 第 2 步：驾驶模式检查
        IF 驾驶模式 != 人工驾驶:
            SET internal_state = PAUSED
            CONTINUE

        // 第 3 步：接收行为观测条目
        IF 收到 ad-07 行为观测条目:
            IF internal_state == EMERGENCY_RO:
                丢弃该条目（紧急状态下漏斗一禁止写入）
                CONTINUE

            SET internal_state = QUERYING

            // 3a. 向世界模型查询当前场景特征
            query_start_time = NOW()
            向 ad-44 发送场景特征查询请求
            WAIT ad-44 返回结果（超时限制 50ms）

            IF 超时 OR 返回异常:
                query_fail_count++
                IF query_fail_count >= 3:
                    上报告警 TO ad-01
                // 降级处理
                SET internal_state = FALLBACK
                场景标签 = "未知道路"
                标签置信度 = 0.3
                GOTO 标签打包步骤

            query_fail_count = 0  // 重置失败计数

            // 3b. 匹配场景标签
            SET internal_state = MATCHING
            场景特征 = ad-44 返回结果
            标签置信度 = 1.0

            // 按优先级从高到低匹配（紧急 > 特殊 > 常规）
            IF 碰撞风险等级 >= 高 OR TTC < 3s:
                场景标签 = "碰撞高风险"
            ELSE IF 检测到特种车辆声光信号:
                场景标签 = "特种车辆临近"
            ELSE IF 路面突发异常检测 == True:
                场景标签 = "路面突发异常"
            ELSE IF 检测到第四类/第三类目标 TTC < 2s:
                场景标签 = "行人/动物突然侵入"
            ELSE IF 天气 ∈ {雨/雪/雾/沙尘} AND 能见度 < 500m:
                场景标签 = "恶劣天气"
            ELSE IF 时段 == 夜间 AND 路面照度 < 10 lux:
                场景标签 = "夜间无灯"
            ELSE IF 交通流密度 > 0.8 AND 平均车速 < 10km/h:
                场景标签 = "拥堵蠕行"
            ELSE IF 路面类型 ∈ {泥土/碎石/沙土}:
                场景标签 = "非铺装道路"
            ELSE:
                场景标签 = "常规路况"

            // 3c. 应急判定
            应急标记 = False
            IF 场景标签 IN ["碰撞高风险", "特种车辆临近", "路面突发异常", "行人/动物突然侵入"]:
                应急标记 = True
            ELSE IF 场景标签 IN ["恶劣天气", "夜间无灯"] AND 行为类型 IN ["紧急制动", "紧急避让"]:
                应急标记 = True

            // 3d. 打包带场景标签的行为条目
            tagged_entry = {
                原始观测条目 = ad-07 行为观测条目,
                场景标签 = 场景标签,
                标签置信度 = 标签置信度,
                应急标记 = 应急标记,
                标记时间戳 = NOW(),
                世界模型查询耗时 = NOW() - query_start_time
            }

            // 3e. 输出到 ad-09
            向 ad-09 发送 tagged_entry
            SET internal_state = NORMAL

        // 第 4 步：周期性状态上报
        IF 距上次上报 >= 5 秒:
            向 ad-01 上报标记状态（含近期各场景标签统计、世界模型查询成功率）

        SLEEP 10ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 世界模型查询超时（> 50ms） | 降级使用“未知道路”标签，置信度 0.3，继续处理 | 世界模型恢复响应，连续 3 次成功后恢复 NORMAL |
| 世界模型返回异常数据（格式错误/字段缺失） | 同超时处理，上报告警 | 世界模型恢复正常输出 |
| 世界模型连续 3 次查询失败 | 上报告警 TO ad-01，后续条目使用“未知道路”标签，不再尝试查询（保护世界模型） | 每 30 秒重试查询一次，成功后恢复 NORMAL |
| 场景特征同时匹配多个标签 | 按优先级取最高者：紧急 > 特殊 > 常规 | — |
| 驾驶模式切换为非人工驾驶 | 暂停处理，状态 PAUSED | 驾驶模式恢复人工驾驶 |
| 紧急熔断 | 停止处理，丢弃未完成条目 | 紧急事件解除 + ad-01 恢复指令 |
| ad-07 行为观测条目数据质量标记为“部分缺失” | 仍正常标记场景标签，但标签置信度降低 0.1 | 数据质量恢复“完整”后置信度恢复 |
| 标签匹配结果与历史同类场景不一致 | 不修正（以世界模型实时输出为准），标记置信度降低 0.05 | — |


## 总线契约

| 总线 | 操作 | 数据内容 | 权限 | 说明 |
|------|------|----------|------|------|
| MemoryBus | 读 | 行为观测条目 | 只读 | 由 ad-07 发送 |
| MemoryBus | 读 | 世界模型场景特征 | 只读 | 由 ad-44 返回 |
| MemoryBus | 读 | 驾驶模式信号 | 只读 | 由 ad-02 下发 |
| MemoryBus | 读 | 紧急熔断指令 | 只读 | 由 ad-01 下发 |
| MemoryBus | 写 | 带场景标签的行为条目 | 专属写入 | 向 ad-09 发送 |
| MemoryBus | 写 | 世界模型查询异常告警 | 事件触发写入 | 向 ad-01 上报 |
| MemoryBus | 写 | 标记状态上报 | 周期性写入 | 每 5 秒向 ad-01 上报 |


## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 漏斗一数据编译期禁止接入自动驾驶决策链路。本模块输出的带标签条目仅流向 ad-09，不向 ECC 大脑或自动驾驶决策链路传输 |
| S-02 | 应急标记为 True 的行为条目，在 ad-09 中自动标记为“应急特殊操作”，不纳入陋习统计 |
| S-03 | 世界模型查询超时或异常时，必须降级使用“未知道路”标签，不可猜测或使用历史缓存标签 |
| S-04 | 场景标签判定以世界模型实时输出为准，不受漏斗记忆经验影响（世界模型外挂隔离） |
| S-05 | 标签置信度低于 0.5 时，ad-09 行为判定应采用保守规则 |
| S-06 | 紧急熔断时立即停止处理并丢弃未完成条目 |

| TC-08-02 | NORMAL 状态，碰撞高风险 | 行为观测条目 + 世界模型返回（TTC=1.5s，碰撞风险=高） | 场景标签=碰撞高风险，应急标记=True |
| TC-08-03 | NORMAL 状态，检测到特种车辆 | 行为观测条目 + 世界模型返回（特种车辆=救护车，声光信号=开启） | 场景标签=特种车辆临近，应急标记=True |
| TC-08-04 | NORMAL 状态，恶劣天气 | 行为观测条目 + 世界模型返回（天气=暴雨，能见度=200m） | 场景标签=恶劣天气，应急标记=False（取决于行为类型） |
| TC-08-05 | NORMAL 状态，世界模型查询超时 | 行为观测条目，世界模型 50ms 内无响应 | 降级，场景标签=未知道路，置信度=0.3 |
| TC-08-06 | 世界模型连续 3 次查询失败 | 第 4 次行为观测条目 | 使用“未知道路”标签，上报告警，不再查询世界模型 |
| TC-08-07 | 驾驶模式切换为自动驾驶 | 行为观测条目 | 暂停处理，状态 PAUSED |
| TC-08-08 | EMERGENCY_RO 状态 | 行为观测条目 | 丢弃条目，不处理 |
| TC-08-09 | 行为条目数据质量=部分缺失 | 行为观测条目 + 世界模型正常 | 正常标记标签，置信度降低 0.1 |
| TC-08-10 | 同时匹配恶劣天气和夜间无灯 | 行为观测条目 + 天气=雨 + 时段=夜间 | 按优先级取“恶劣天气”（天气优先于时段） |
