# mem-64 驾驶员身份识别单元

> 所属仓库: AD-mlnf-mem/spec/
> 模块编号: mem-64
> 所属分区: AD新增·双漏斗管理层


## 一、基本信息

| 项目 | 内容 |
|:---|:---|
| 模块编号 | mem-64 |
| 模块名称 | 驾驶员身份识别单元 |
| 核心职责 | 统一管理中控屏手动选择、座椅记忆联动、人脸识别三种身份识别方式。车辆启动时提示选择驾驶员身份，支持行程中随时切换 |
| 输入数据 | 中控屏触控选择信号、座椅位置调节数据（来自座椅记忆模块）、车内摄像头面部特征数据（可选，来自车内视觉模块）、车辆启动信号 |
| 输出数据 | 驾驶员身份确认信号（含子画像槽ID，发送至mem-61）、身份识别方式标记、识别置信度、未识别超时信号 |


## 二、核心处理逻辑

**1. 身份识别触发**

车辆上电启动时，系统自动触发驾驶员身份识别流程。行程中用户也可通过中控屏随时手动触发身份切换。

**2. 三种识别方式**

系统按以下优先级依次尝试：

**优先级一：中控屏手动选择**

车辆启动后，中控屏弹出“您好，请问是谁在开？”提示界面。界面列出已注册的所有长期子画像槽（显示槽位名称，如“张三”、“李四”），以及“临时使用”选项。用户点击选择后，立即确认身份，准确率100%。

**优先级二：座椅记忆联动**

用户调节座椅时触发座椅记忆模块的位置匹配。若当前座椅位置与已存储的座椅记忆位置（如“张三”的座椅位置）匹配度超过预设阈值，系统在中控屏提示“检测到座椅位置与[张三]匹配，是否确认？”。用户确认后完成身份切换，用户取消则回退到手动选择界面。

**优先级三：人脸识别（可选，默认关闭）**

若用户已在设置中开启人脸识别功能，车内摄像头扫描驾驶位面部特征，与已注册的面部特征进行匹配。匹配成功后在中控屏提示确认。人脸识别功能可随时在设置中关闭。关闭后系统不采集、不存储任何面部特征数据。

**3. 超时处理**

若5秒内无任何选择、无座椅记忆匹配信号、无人脸识别匹配信号，系统默认启用临时画像槽。临时画像槽在7天后自动清除。行程中可随时通过中控屏切换至已注册的长期槽。

**4. 临时画像槽与一次性记录槽**

临时画像槽：短期借用者使用，数据保留7天后自动清除。一次性记录槽：代客泊车等场景使用，行程结束（车辆熄火+P档）后立即彻底清除。一次性记录槽的数据不执行7天保留，直接执行DoD 5220.22-M标准覆写擦除。

**5. 隐私保护**

人脸识别功能默认关闭，需用户主动在设置中开启。人脸特征数据仅本地加密存储，物理隔绝云端同步通道。用户删除子画像槽时，关联的人脸特征数据同步彻底擦除。座椅记忆数据同样仅本地存储。


## 三、约束与异常处理

| 约束类型 | 内容 |
|:---|:---|
| 识别超时 | 5秒内无任何识别结果，默认启用临时画像槽 |
| 人脸识别 | 默认关闭，需用户主动开启；关闭后不采集任何面部数据 |
| 隐私保护 | 所有生物特征数据（人脸）仅本地加密存储，永不离开本地 |
| 数据擦除 | 子画像槽删除时关联的座椅记忆和人脸数据同步擦除 |
| 一次性槽 | 行程结束立即清除，不保留 |
| 模式限制 | 自动驾驶模式下禁止切换驾驶员身份，须先切换至人工驾驶模式 |


## 四、接口校验用例

**用例1：中控屏手动选择**

前置条件：已注册长期槽“张三”和“李四”。操作：车辆启动，中控屏弹出选择界面，用户点击“张三”。预期：mem-64输出身份确认信号（目标槽ID=张三），发送至mem-61激活对应子画像槽。

**用例2：座椅记忆联动**

前置条件：已存储“张三”的座椅位置。操作：车辆启动，用户调节座椅至与“张三”存储位置匹配度超过阈值。预期：中控屏弹出“检测到座椅位置与[张三]匹配，是否确认？”，用户确认后完成身份切换。

**用例3：5秒超时默认临时槽**

前置条件：车辆启动，未存储任何座椅记忆，人脸识别关闭。操作：5秒内中控屏无任何操作。预期：系统默认启用临时画像槽。中控屏提示“已启用临时驾驶模式，行程数据将在7天后自动清除”。

**用例4：人脸识别关闭时隐私保护验证**

前置条件：用户已在设置中关闭人脸识别。操作：系统启动，执行身份识别流程。预期：车内摄像头不启动面部扫描功能，系统不采集、不存储任何面部特征数据。仅通过中控屏手动选择或座椅记忆联动完成身份识别。

**用例5：一次性记录槽使用**

前置条件：用户在中控屏选择“代客泊车模式”。操作：车辆行驶至目的地，用户熄火+P档。预期：本次行程的所有驾驶数据立即彻底清除，不保留。下次启动时系统弹出正常的身份选择界面。
