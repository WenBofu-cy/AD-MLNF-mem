# mem-67 交通法规库

> 所属仓库: AD-mlnf-mem/spec/
> 模块编号: mem-67
> 所属分区: 独立外置模块


## 一、基本信息

| 项目 | 内容 |
|:---|:---|
| 模块编号 | mem-67 |
| 模块名称 | 交通法规库 |
| 核心职责 | 存储法定通行刚性规则，以“场景条件-强制执行动作”形式固化，优先级高于一切驾驶经验。只读不改，永不遗忘，不参与任何漏斗晋升与遗忘机制 |
| 输入数据 | 法规查询请求（来自AD-ECC交通法规校验模块adecc-13）、当前驾驶场景特征（场景类别/道路属性/交通标识状态/周边目标分布） |
| 输出数据 | 适用法规条目列表（规则ID/规则描述/执行刚性等级/强制性动作）、法规冲突消解结果（当多条法规同时触发时，按优先级排序后的最终动作） |


## 二、核心处理逻辑

**1. 法规存储结构**

每条法规以“场景条件-强制执行动作”形式固化：

- 规则ID：唯一标识，格式“TR-XXX”
- 规则描述：人类可读的法规内容
- 触发条件：场景特征匹配条件（如“检测到红灯信号”）
- 强制执行动作：条件满足时必须执行的动作（如“停车等待”）
- 执行刚性等级：硬约束（不可违抗）/ 软约束（结合场景判断）/ 软追加（需人工审核）

**2. 法规查询与匹配**

adecc-13（交通法规校验）向mem-67发起法规查询请求，携带当前驾驶场景特征。mem-67检索所有触发条件匹配当前场景特征的法规条目，按执行刚性等级排序返回。硬约束条目优先于软约束条目。若多条硬约束同时触发且动作存在冲突，按法规内置的优先级排序（如“人行横道行人优先”优先于“直行优先”）。

**3. 法规硬约束保护**

标记为“硬约束”的法规条目，物理不可违抗。这些条目不经过AD-ECC的行为决策推演评估，直接作为不可修改的前提条件注入决策流程。任何自主学习经验、驾驶员习惯、时效优先策略均不得突破法规硬约束。

**4. 法规更新机制**

法规库仅接受官方离线更新。更新包须校验数字签名，不匹配则拒绝。更新为追加写入模式，不改动既有核心规则。地方临时交通管制、阶段性新规等，以软追加方式录入，需人工审核后方可生效。

**5. 法规冲突消解**

当同一场景触发多条法规且动作存在矛盾时，按以下优先级消解：

- 人身安全优先（行人保护 > 车辆保护）
- 法定强制优先（硬约束 > 软约束）
- 特殊法优先于一般法（特种车辆优先规则 > 一般路权规则）
- 即时危险优先（紧急避险规则 > 一般通行规则）

消解结果随决策摘要同步记录，可追溯。


## 三、约束与异常处理

| 约束类型 | 内容 |
|:---|:---|
| 只读保护 | 法规库数据不可被任何经验、习惯、学习行为修改或覆盖 |
| 更新权限 | 仅接受官方离线更新，须通过数字签名校验 |
| 硬约束优先级 | 硬约束条目物理不可违抗，不作为推演评估的对象，直接作为决策前提条件 |
| 追加写入 | 新法规仅追加写入，不改动既有核心规则 |
| 软追加审核 | 软追加条目须人工审核后方可生效 |
| 冲突消解可追溯 | 所有冲突消解结果写入决策摘要，存储周期≥6个月 |


## 四、接口校验用例

**用例1：红灯场景法规查询**

输入：当前场景特征包含“检测到红灯信号，车辆处于直行车道”。预期：返回硬约束条目“红灯必须停车”，执行动作“在停止线前停车等待”，刚性等级=硬约束。

**用例2：人行横道行人优先**

输入：当前场景特征包含“检测到人行横道，有行人正在横穿”。预期：返回硬约束条目“人行横道行人绝对优先”，执行动作“停车让行”。该条目优先级高于任何其他路权规则。

**用例3：经验不可覆盖法规**

前置条件：漏斗二中存储了一条经验“该路口夜间车流极少，多数车辆选择减速通过”。操作：AD-ECC尝试以该经验为依据，在红灯场景下决策“减速通过”。预期：adecc-13查询mem-67后返回硬约束“红灯必须停车”。该硬约束直接否决经验驱动的决策，强制执行停车。

**用例4：法规冲突消解**

输入：当前场景同时触发两条规则——“特种车辆优先：检测到救护车声光信号，立即执行主动让行”和“红灯必须停车”。预期：冲突消解按“人身安全优先”原则，救护车承载人命救助义务，其优先权高于红灯停车规则。最终输出“在确保安全的前提下，主动让行”。

**用例5：法规更新数字签名校验**

操作：系统接收到一份未通过数字签名校验的“法规更新包”。预期：mem-67拒绝该更新包，记录异常事件至mem-55日志系统。既有法规数据不受任何影响。
