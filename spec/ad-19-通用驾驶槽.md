# ad-19-通用驾驶槽 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-19 |
| 模块名称 | 通用驾驶槽 |
| 所属分区 | 三、漏斗二：自动驾驶自成长漏斗 / 场景分槽管理 |
| 核心职责 | 承载跨场景通用驾驶风格、乡村道路、未匹配到专属分槽的日常驾驶经验的完整五层存储与晋升管理。作为漏斗二的兜底分槽，确保任何无法精确归类的驾驶经验都有归宿。乡村道路子类享有独立遗忘保护参数。是本场景经验从 L1 到 L5 的完整生命周期管理者 |
| 依赖模块 | ad-14（场景判定与分槽路由单元，接收经验写入指令）、ad-20 至 ad-30（五层存储与晋升遗忘执行模块）、ad-36（综合重要度 I 值聚合计算单元） |
| 被依赖模块 | ad-14（上报存储占用率与活跃状态）、ad-03（漏斗二专属调度单元，接收状态汇总） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常运行 | `NORMAL` | 正常接收经验条目并执行五层存储流转 | 漏斗二已激活，本槽已初始化 |
| 容量告急 | `CAPACITY_WARNING` | 存储占用率 > 90%，已触发温和加速遗忘 | 本槽存储占用率监测 |
| 维护中 | `MAINTENANCE` | 正在执行相似经验归并或冷归档操作 | 定期维护周期或容量告急触发 |
| 冻结只读 | `FROZEN` | 漏斗二冻结或紧急接管，暂停写入但保留查询 | 驾驶模式切换或紧急熔断 |


## 输入数据

| 输入项 | 数据类型 | 来源模块 | 触发条件 | 优先级 |
|--------|----------|----------|----------|:---:|
| 经验写入指令 | Struct（含经验条目 + 写入优先级 + 路由标记 + 判定置信度） | ad-14 场景判定与分槽路由单元 | 每次场景路由至本槽时 | **高** |
| L1 至 L4 晋升候选清单 | List（经验条目 ID + 晋升目标层级 + 重要度 I） | ad-38 晋升双条件判定单元 | 每个晋升判定周期 | **高** |
| 遗忘候选清单 | List（经验条目 ID + 当前层级 + I 值） | ad-40 遗忘阈值判定单元 | 每个遗忘判定周期 | **高** |
| 归并建议 | Struct（源经验 ID + 目标经验 ID + 相似度） | ad-25 L3 中期层相似经验归并单元 | L3 归并操作时 | 普通 |
| 全局调度指令 | Enum（冻结/激活/维护） | ad-03 漏斗二专属调度单元 | 模式切换时 | **高** |

## 专属遗忘策略

### 一、常规通用子类

| 策略项 | 标准值 | 本子类调整 | 调整理由 |
|--------|:---:|:---:|----------|
| 全部权重、阈值、晋升时间 | — | 使用漏斗二标准默认值 | 通用场景无特殊偏向，保持均衡 |

### 二、乡村道路子类

| 策略项 | 标准值 | 本子类调整 | 调整理由 |
|--------|:---:|:---:|----------|
| 最低重要度遗忘阈值 | 0.15 | **0.075（下调 50%）** | 乡村道路经验重复频率极低但安全价值高，须长期保留 |
| 安全显著性权重（α） | 0.5 | 0.55 | 乡村道路行人/牲畜/坑洼突发风险高 |
| 复用频次权重（γ） | 0.3 | 0.20 | 同一条乡村道路可能数月才走一次 |
| 风格匹配度权重（β） | 0.2 | 0.25 | 乡村道路舒适度是重要体验维度 |
| L1→L2 晋升时间阈值 | 24h | 36h（延长） | 乡村驾驶经验需要更长观察期 |
| L2→L3 晋升时间阈值 | 7日 | 10日（延长） | 低频场景，延长固化周期 |
| 遗忘处理方式 | 直接删除 | **优先冷归档** | 乡村道路经验宁可归档也不丢弃 |


## 核心处理逻辑

```
FUNCTION general_driving_slot_main_loop():
    SET internal_state = NORMAL
    初始化五层存储结构（配额按漏斗二标准比例分配）
    加载常规子类策略（使用标准默认参数）
    加载乡村道路子类策略（独立遗忘保护参数）
    初始化子类标记字典（经验条目 ID → 子类标记）

    WHILE 系统运行中:
        // 第 1 步：全局调度
        IF 收到 ad-03 冻结指令:
            SET internal_state = FROZEN
            CONTINUE
        ELSE IF 激活指令:
            SET internal_state = NORMAL
            CONTINUE

        // 第 2 步：接收新经验并区分处理
        IF 收到经验写入指令 AND internal_state == NORMAL:
            经验条目 = 写入指令.经验条目
            路由标记 = 写入指令.路由标记

            IF 路由标记 == "乡村道路子类":
                经验条目.I₀ = MIN(经验条目.I₀ + 0.05, 1.0)
                子类标记字典[经验条目.ID] = "乡村道路"
            ELSE IF 路由标记 == "低置信度降级" OR 路由标记 == "世界模型降级":
                子类标记字典[经验条目.ID] = "降级路由"
                经验条目.I₀ = 经验条目.I₀（不变，保留原始）
            ELSE IF 路由标记 == "归并溢出":
                子类标记字典[经验条目.ID] = "归并溢出"
                经验条目.I₀ = MIN(经验条目.I₀ + 0.03, 1.0)
            ELSE:
                子类标记字典[经验条目.ID] = "常规通用"

            写入 L1
            向 ad-14 返回存储完成回执

        // 第 3 步：晋升处理（按子类区分阈值）
        IF 收到晋升候选清单:
            FOR EACH 候选 IN 清单:
                子类 = 子类标记字典[候选.经验条目.ID]
                IF 子类 == "乡村道路":
                    使用乡村道路子类晋升时间阈值
                ELSE:
                    使用标准晋升时间阈值

                调用 ad-39 搬运
                向 ad-38 返回成功/失败

        // 第 4 步：遗忘处理（按子类区分策略）
        IF 收到遗忘候选清单:
            FOR EACH 候选 IN 清单:
                IF 候选.层级 IN [L4, L5]:
                    SKIP
                子类 = 子类标记字典[候选.经验条目.ID]
                IF 子类 == "乡村道路":
                    遗忘阈值 = 0.075
                    IF 候选.I < 遗忘阈值:
                        调用 ad-42 归档（冷归档，不删除）
                        向 ad-40 返回已归档
                    ELSE:
                        向 ad-40 返回保留
                ELSE:
                    遗忘阈值 = 0.15（标准）
                    IF 候选.I < 遗忘阈值:
                        调用 ad-42 删除
                        向 ad-40 返回已删除
                    ELSE:
                        向 ad-40 返回保留

        // 第 5 步：L3 归并（谨慎处理，不同子类不混归）
        IF 距上次归并 >= 24h OR 占用率 > 85%:
            SET internal_state = MAINTENANCE
            向 ad-25 发送归并请求（附加约束：不同子类不合并）
            执行归并（相似度阈值=0.7，常规值）
            SET internal_state = NORMAL

        // 第 6 步：容量监控
        占用率 = 计算占用率()
        IF 占用率 > 0.90:
            SET internal_state = CAPACITY_WARNING
            上报告警
            触发温和加速遗忘：仅对 L1/L2 常规子类临时上调遗忘阈值 20%
        ELSE IF 占用率 < 0.70 AND internal_state == CAPACITY_WARNING:
            SET internal_state = NORMAL
            恢复标准遗忘阈值

        // 第 7 步：定期乡村道路子类统计
        IF 距上次统计 >= 60 秒:
            统计乡村道路子类条目数、各层级分布、平均 I 值
            向 ad-03 上报

        // 第 8 步：状态上报
        IF 距上次上报 >= 1 秒:
            向 ad-14、ad-03 上报槽状态快照

        SLEEP 100ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 降级路由经验长期积压 | 不做特殊处理，正常参与晋升与遗忘 | — |
| 乡村道路子类经验被标准遗忘误删 | 不会发生（乡村子类遗忘阈值 0.075，归档而非删除） | — |
| 不同子类的经验被合并 | 归并时附加约束禁止跨子类合并，若误合并则回滚并告警 | 手动修复 |
| 通用驾驶槽作为兜底槽被删除 | 编译期禁止删除通用驾驶槽，任何删除请求均拒绝 | — |
| 容量告急但乡村道路子类占用高 | 不处理乡村道路子类，仅加速遗忘常规子类 L1/L2 | 容量释放 |
| 冻结状态写入 | 拒绝，返回原因 | 激活指令 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 通用驾驶槽作为漏斗二的兜底分槽，编译期保证其始终存在，任何运行时进程不可删除或禁用 |
| S-02 | 乡村道路子类经验遗忘阈值 0.075 为硬编码下限，不可进一步下调 |
| S-03 | 乡村道路子类经验遗忘方式硬编码为“冷归档”，不可直接删除 |
| S-04 | L3 归并时禁止跨子类合并，防止乡村道路经验与通用经验混淆 |
| S-05 | 容量告急时加速遗忘仅针对常规子类 L1/L2，乡村道路子类不受影响 |
| S-06 | 所有晋升、遗忘、归并操作全量写入 ad-51 变更日志 |
