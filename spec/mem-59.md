# mem-59 独立世界模型库

> 所属仓库: AD-mlnf-mem/spec/
> 模块编号: mem-59
> 所属分区: 外置独立扩展区（AD场景适配）


## 一、基本信息

| 项目 | 内容 |
|:---|:---|
| 模块编号 | mem-59 |
| 模块名称 | 独立世界模型库 |
| 核心职责 | 存储驾驶场景相关的物理常识与结构化世界模型。包含道路物理属性、交通参与者属性、非结构化道路参数、环境动态适配参数。以结构化属性向量与IF-THEN因果规则表形式存储，只读不改，永不遗忘，不参与任何漏斗晋升与遗忘机制 |
| 输入数据 | 世界模型更新请求（来自adecc-03局部场景建模）、环境查询请求（来自AD-ECC各认知模块及AD-MCC各执行模块） |
| 输出数据 | 物体属性基底（硬度/脆性/摩擦系数等量化等级）、物理约束规则、实时环境图谱、场景先验数据、路面类型与对应的摩擦系数和制动距离系数 |


## 二、核心处理逻辑

**1. 物理常识存储结构**

以结构化属性向量与IF-THEN因果规则表形式存储，存储占用<300KB。

道路实体物理参数：五大类目标的材质硬度、摩擦系数、质量等级、形变特性、可移动性。典型条目：干燥沥青μ=0.8，压实积雪μ=0.2，冰面μ=0.1。

环境物理影响阈值：积水滑水临界车速、冰雪附着系数衰减曲线、能见度-安全车距映射。典型条目：水膜厚度>3mm且车速>80km/h触发滑水风险。

车辆动力学硬边界：最大侧向加速度0.85g、极限制动减速度1.0g、侧翻临界阈值。

环境动态适配参数：随天气、光照、时段自动联动风险判定权重的修正系数。典型条目：夜间无灯路段，行人出现概率权重×1.5。

车辆能源物理参数：电池额定容量、燃油箱容积、额定续航、各工况能耗曲线。

**2. 非结构化道路物理属性**

| 属性 | 典型值 | 对驾驶的影响 |
|:---|:---|:---|
| 压实泥土路面摩擦系数 | μ=0.5-0.6 | 制动距离较沥青路面延长约30% |
| 松散碎石路面摩擦系数 | μ=0.3-0.4 | 急刹易侧滑，须降低车速上限 |
| 湿泥路面摩擦系数 | μ=0.2-0.3 | 极度危险，禁止急刹急转 |
| 路面可通过性判断 | 坑洼深度>15cm或凸起>20cm | 触发绕行规划，不可强行通过 |
| 道路宽度判断 | 单车道<2.5m | 不具备会车条件，须找避让点 |

**3. 交通参与者属性模板**

五大类交通参与者的物理属性预置模板：

- 行人（含儿童/老人/行动不便者）：可移动、脆性5（极高）、质量1（极轻）、避让优先级最高
- 非机动车（自行车/电动车）：可移动、速度范围0-25km/h、脆性4
- 普通机动车（乘用车/商用车）：可移动、质量3-5、速度范围0-120km/h、硬度4-5
- 静态障碍物（护栏/隔离带/路沿/坑洼）：不可移动、硬度4-5、脆性1-4
- 特种车辆（救护车/消防车/警车）：可移动、声光特征、优先通行权标记

**4. 查询与更新机制**

AD-ECC各认知模块通过MemoryBus向mem-59发起查询请求，获取当前场景相关的物理常识和世界模型数据。adecc-03（局部场景建模）负责将感知数据转化为结构化世界模型，增量更新至mem-59。更新内容须通过物理一致性校验（如“路面摩擦系数应在0.1-1.0范围内”），校验未通过则拒绝写入。世界模型数据与MLNF-Mem记忆漏斗物理隔离，互不干扰。
